Advanced image optimization plugin with AI integration for WordPress. Optimize images with local processing (GD/ImageMagick) and external AI services.

== Description ==

Kafunel Optimizer AI is a powerful image optimization plugin that combines local processing with advanced AI technology to dramatically reduce image file sizes while maintaining quality. The plugin supports all major image formats and offers both free and Pro features.

= Key Features: =

* Support for JPEG, PNG, GIF, and WebP input formats
* Output to JPEG, PNG, WebP, and AVIF formats
* Multiple compression levels: Lossless, Light (Optimal), Aggressive, Maximum
* Automatic image resizing with customizable dimensions
* Intelligent conversion to modern formats (WebP/AVIF)
* Local optimization using GD Library or ImageMagick
* Optional AI-powered optimization for superior results
* Bulk optimization in media library
* Detailed optimization statistics and savings reports
* Automatic backup of original images
* Free plan with daily limits and Pro upgrade option

= Free Plan Features: =

* Optimize up to 10 images per day
* Batch optimization up to 20 images at a time
* Basic compression with local processing
* Support for all major image formats

= Pro Plan Features: =

* Unlimited daily optimizations
* Advanced AI-powered compression
* Background removal using AI
* AI-powered image upscaling
* Detailed analytics and reporting
* Priority support
* No optimization limits

== Installation ==

1. Upload the `kafunel-optimizer-ai` folder to the `/wp-content/plugins/` directory
2. Activate the plugin through the 'Plugins' menu in WordPress
3. Go to Settings > Kafunel Optimizer to configure your settings
4. Add your API key if you have a Pro subscription (optional for local optimization)
5. Start optimizing your images in the Media Library

== Frequently Asked Questions ==

= What image formats are supported? =

The plugin supports JPEG, PNG, GIF, and WebP as input formats. Output formats include JPEG, PNG, WebP, and AVIF.

= Do I need an API key? =

An API key is optional. The plugin works with local optimization using GD Library or ImageMagick. API keys unlock advanced AI features available in the Pro plan.

= Is there a free version? =

Yes, the plugin includes a free version with basic optimization features and daily limits. Pro features require a subscription.

= Can I convert images to WebP or AVIF? =

Yes, the plugin can convert images to modern formats like WebP and AVIF for better performance.

= Will optimization affect image quality? =

The plugin offers multiple compression levels. Lossless and Light (Optimal) settings preserve quality while reducing file size. Aggressive and Maximum settings provide higher compression with some quality trade-offs.
